## How to Run:

### Option 1: With Database (Recommended)
```bash
npm install
node server-with-db.js
```
Visit: http://localhost:3000

### Option 2: Simple Server
```bash
node server.js
```
Visit: http://localhost:3000

### Admin Panel
Visit: http://localhost:3000/admin
